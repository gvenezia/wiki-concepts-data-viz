# Web scraper
This web scraper is designed to find similar concepts then display them in a dendrogram (a tree-like structure for showing hierarchy).

### How was it built?
- Node
- D3  
- cheerio
- Canvas
- fs

### How to View  
1. Clone the repo to your computer  
2. `python -m SimpleHTTPServer [port#]` in the root folder  
3. Visit the port number and click on which type of dendrogram you'd like to view
